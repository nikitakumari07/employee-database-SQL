A structured SQL practice project based on an employee management database.  This repository demonstrates CRUD operations, sorting, filtering, and aggregate functions  using MySQL. It is created for learning, practice, and interview preparation purposes.

# Company SQL Practice

This project is a structured SQL practice based on an employee database.

## Project Overview
The repository demonstrates basic to intermediate SQL concepts using MySQL.

It includes:
- Database creation
- Table creation
- Data insertion
- SELECT queries
- Filtering with WHERE
- Sorting using ORDER BY
- LIMIT clause
- UPDATE operations
- DELETE operations
- Aggregate functions (MIN, MAX, SUM, AVG, COUNT)
